\doxysubsection{Opis szczegółowy}
Główny silnik grafiki 3D. 

Singleton zarządzający całym systemem renderowania Open\+GL. Odpowiada za inicjalizację, pętlę główną, obsługę zdarzeń oraz zarządzanie sceną 3D i obiektami.
