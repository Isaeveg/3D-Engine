\doxysubsection{Opis szczegółowy}
Kamera do obserwacji sceny 3D. 

Klasa \doxylink{class_camera}{Camera} implementuje system kamery perspektywicznej z obsługą ruchu klawiatury i myszy. Wykorzystuje macierze widoku Open\+GL do transformacji sceny.
