\doxysubsection{Opis szczegółowy}
Menedżer tekstur do renderowania Open\+GL. 

Klasa \doxylink{class_bitmap_handler}{Bitmap\+Handler} odpowiada za ładowanie plików obrazów (BMP, PNG, JPG) i konwersję ich na tekstury Open\+GL. Obsługuje również wiązanie i odwiązanie tekstur podczas renderowania.
