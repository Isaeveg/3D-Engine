Opis szczegółowy

Bazowa klasa dla wszystkich obiektów w scenie 3D.

Klasa GameObject stanowi fundament dla każdego obiektu, który ma być rysowany w przestrzeni 3D. Przechowuje informacje o pozycji, rotacji, skali i kolorze obiektu.
