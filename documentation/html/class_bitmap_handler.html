Opis szczegółowy

Menedżer tekstur do renderowania OpenGL.

Klasa BitmapHandler odpowiada za ładowanie plików obrazów (BMP, PNG, JPG) i konwersję ich na tekstury OpenGL. Obsługuje również wiązanie i odwiązanie tekstur podczas renderowania.
